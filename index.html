My Projects
Dibs

Dibs is an online platform that lets members in campus communities share reusable items among with other members in those communities. The ultimate goal of this project is to promote campus-wide sustainability awareness and waste reduction.

Built with: Django (Python web framework)

Collaborators: Ken Ruto, Vincent Yu

Chatter

A simple Django-based group chat application which saves chat history. My goal is to make it a reusable Django package and release it on PyPi for other developers building Django apps and looking to integrate a chat functionality into those apps.

Built with: Django (Python web framework), Django Channels 2, Redis

Next Blue Bus

A simple web app that lets students at Haverford and Bryn Mawr Colleges check the departure times of the colleges' shuttle buses.

Built with: Flask (Python Microframework)

Collaborator: Ken Ruto
